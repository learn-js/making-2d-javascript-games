Who is this for?

The ideal reader for this book is someone who likes exploring, imagining, and inventing for themselves. You probably have some experience with javascript already, and you'd like to learn more about animation using the canvas tag, basic game development patterns, and gain intermediate skills in developing javascript modules that can be used on the server and in the browser.

What if I want to learn the basics of JavaScript?

This book assumes some familiarity with JavaScript, but still provides a gentle introduction to some of the details of the language.

You should also check out our book Introduction to JavaScript & Node.js for a basic intro to the language.

Also consider learning more at codecademy.com and with the book Eloquent JavaScript.
